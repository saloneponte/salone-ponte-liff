## 🚀 開発サーバー起動方法

### プロジェクトナビゲーション
```bash

# http://localhost:3000
```

```bash
npm run dev:liff  
# http://localhost:3001/index.html
```

### 管理ダッシュボード
```bash
npm run dev:dashboard
# http://localhost:3002/dashboard.html
```

## ⚠️ よくある問題と解決方法

### CSS/JS読み込みエラー
**症状**: スタイルが適用されない、JavaScriptが動作しない
**原因**: 相対パスの間違い
**解決**: `../styles/`, `../scripts/` のパス確認

### LIFF初期化エラー
**症状**: Console に "LIFF初期化エラー" 表示
**原因**: LIFF環境外での実行（正常動作）
**解決**: デモモードで動作するため問題なし

### Firebase接続エラー
**症状**: Firestore読み込みエラー
**原因**: Firebase設定の問題
**解決**: `src/config/firebase.js` の設定確認

### 定数未定義エラー
**症状**: `LIFF_CONFIG is not defined`
**原因**: script読み込み順序の問題
**解決**: constants.js を他のファイルより先に読み込み
